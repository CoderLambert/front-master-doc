# 实战项目：购物车应用 (Shopping Cart)

import { CodePlayground, LiveCode } from '../../src/components';

## 项目概述

购物车应用是学习React状态管理的经典项目。本项目将帮助您掌握：

- 复杂对象的状态管理
- 数组的增删改查操作
- 价格计算和格式化
- 条件渲染
- Context API 的使用
- useReducer Hook 的使用

## 功能需求

我们的购物车应用将包含以下功能：

1. 🛍️ 商品列表展示
2. ➕ 添加商品到购物车
3. ➖ 增减商品数量
4. 🗑️ 删除商品
5. 💰 计算总价
6. 🛒 购物车商品统计
7. 📦 结账功能

    const existingItem = cart.find(item => item.id === product.id);

    if (existingItem) {
      setCart(cart.map(item =>
        item.id === product.id
          ? { ...item, quantity: item.quantity + 1 }
          : item
      ));
    } else {
      setCart([...cart, { ...product, quantity: 1 }]);
    }

      return item;

    setCart(cart.filter(item => item.id !== id));

## 学习要点

1. **复杂状态管理** - 管理商品和购物车两种状态
2. **数组操作** - 使用 map、filter、reduce 方法
3. **条件渲染** - 根据购物车状态显示不同内容
4. **事件处理** - 处理点击和数量变化事件
5. **数据计算** - 实时计算总价和商品数量
